
# Adventure game

## Description

CLI based text adventure game, where the player can move betweem areas on a map
and collect items within the areas. Game is still in prototype phase and does
not yet have a solid goal or quest in it.

## Motivation

Project was made as a part of a school DevOps course. Original idea for the game
was a final project for a Python course and is being further developed in this
repository.
